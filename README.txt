Sistema de Gerenciamento de Cursos de Idiomas

Descrição:

Este é um sistema simples para gerenciar a inscrição de alunos em cursos de idiomas. O sistema permite que novos alunos escolham um dos três cursos disponíveis (Inglês, Espanhol e Francês), garantindo um limite de até duas turmas por curso, com um máximo de 20 alunos por turma.

---------------------------------------------------------------------------------------------------------------------------------

Funcionalidades
- Cadastrar alunos com nome, data de nascimento e idioma escolhido.
- Distribuir os alunos entre as turmas disponíveis.
- Listar os alunos cadastrados em cada curso e turma.
- Menu interativo via terminal.

---------------------------------------------------------------------------------------------------------------------------------

Tecnologias Utilizadas
- Linguagem: Python 3
- Programação Orientada a Objetos (POO)
- Módulo `datetime` para validação de datas

---------------------------------------------------------------------------------------------------------------------------------

Como Executar o Programa
1. Certifique-se de ter o Python 3 instalado.
2. Baixe o arquivo do código-fonte.
3. Execute o arquivo.
4. Siga as instruções no menu interativo.

---------------------------------------------------------------------------------------------------------------------------------

Estrutura do Código
  O código está organizado em três classes principais:
    - Aluno: Representa um aluno, armazenando nome, data de nascimento e idioma escolhido.
    - Curso: Gerencia turmas para um idioma específico, garantindo o limite de alunos por turma.
    - EscolaIdiomas: Gerencia os cursos disponíveis e interage com o usuário para cadastrar alunos e exibir informações.
